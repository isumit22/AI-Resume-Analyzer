# Frontend

React + Vite UI for resume upload, ATS scoring, keyword gaps, recruiter feedback, and a learning roadmap.

## Run

1. Install dependencies with `npm install`.
2. Start the app with `npm run dev`.

## Notes

- `VITE_API_URL` should point to the FastAPI backend, for example `http://localhost:8000`.
- The UI is styled with Tailwind CSS and structured so shadcn/ui can be added next.
